# swisstool

Demo project for experience gaining in React ecosystem ([Live demo here][demo]).

## Planned tools
- Dashboard
- Todo-list (obviously)
- Notes
- Pomodoro timer

## Technology stack
- SASS / SCSS
- React
- Redux
- Webpack

## Build project

1. Install dependencies
```bash
npm install
```

2. Serve project on your local machine
```bash
npm run start
```

3. Build project
```bash
npm run build
```
